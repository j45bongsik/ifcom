기관 사용자 정보 관리

  • 아이디 변경은 사용자가 사용 권한 신청 시 오입력한 것을 수정하는 것 이외에는 변경을 금합니다.
  • 아이디 변경과 일회용 비밀번호 전송 기능을 제외한 기타 정보 수정 시 페이지 하단의 ‘정보 수정’ 버튼을 클릭해야 수정이 완료됩니다.
  • 사용자의 모든 상태 관리는 ‘상태 변경’ 클릭 후 출력되는 팝업에서 처리(사용 승인, 접속 불가 해제 , 휴면 상태 관리 등) 가능합니다.
  • 권한 승계는 사용자의 요청이 있는 경우에만 진행해 주세요.
